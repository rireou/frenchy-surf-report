# Netlify preview workflow verification

This documentation-only change verifies the safe publishing workflow for the Frenchy surf report:

1. ChatGPT/Codex creates a branch and draft pull request.
2. Netlify builds a separate Deploy Preview for that pull request.
3. Robin reviews the preview before approving the change.
4. Only merging the pull request publishes the update to the live surf report.

Created on 15 July 2026. This file does not change the public surf report.
